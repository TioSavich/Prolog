# Misconceptions Corpus — Pattern Detection Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Encode 1,424 in-scope research-literature misconceptions as Prolog rules, run each against known correct answers, and classify as `wrong_answer`, `loop_detected`, or `undefined` — using only native Prolog arithmetic, with theoretical annotations as comments.

**Architecture:** Lights-on/lights-off: every file has a NATIVE LAYER (standard Prolog `is/2`, always runs) and a GROUNDED LAYER (commented-out calls to grounded arithmetic, switched in later). A shared test harness runs all registered misconceptions, writes `misconceptions/results.csv`, and prints a summary. Axioms are certified before misconception encoding begins.

**Tech Stack:** SWI-Prolog (test runner pattern: `swipl -l paths.pl -l <file> -g run_tests -t halt`), SQLite3 (research.db), Python 3 (DB triage scripts).

This makes M5-Direct identical to M5 in its premise set, which is intentional — the unguarded version was firing spuriously on arbitrary ground lists.

### The two registration conventions

**Arithmetic misconceptions** (fraction, decimal, whole_number, measurement):
```prolog
arith_misconception(
    Source,      % db_row(ID:integer) or asktm(Code:atom)
    Domain,      % fraction | decimal | whole_number | measurement
    Description, % atom: short snake_case name
    RuleName,    % atom: name of the Prolog predicate to call
    Input,       % ground term passed as first arg to RuleName
    Expected     % ground term the CORRECT rule would return
).
```
The harness calls `call(RuleName, Input, Got)` via `call_with_inference_limit/3`.

**Entailment misconceptions** (geometry):
```prolog
entail_misconception(
    Source,       % db_row(ID) or asktm(Code)
    Description,  % atom: short name
    Shape,        % atom: shape being claimed
    Target,       % atom: shape the student claims Shape entails
    Claim         % holds | fails — what the student asserts about entailment
).
```
The harness checks `entails_via_incompatibility(Shape, Target)` against the claim.

```prolog
% Misconception Test Harness
% Run: swipl -l paths.pl -l misconceptions/test_harness.pl -g run_all -t halt
%
% Loads all misconception modules, runs every registered check,
% prints a summary, and writes results.csv.

:- module(test_harness, [run_all/0, classify_arith/5, classify_entail/4]).

:- use_module(library(lists)).
:- use_module(formalization(axioms_geometry), [entails_via_incompatibility/2]).

% Load all misconception modules
:- use_module(misconceptions(misconceptions_fraction)).
:- use_module(misconceptions(misconceptions_whole_number)).
:- use_module(misconceptions(misconceptions_decimal)).
:- use_module(misconceptions(misconceptions_measurement)).
:- use_module(misconceptions(misconceptions_geometry)).

%! classify_arith(+RuleName, +Input, +Expected, -Class, -Got) is det.
classify_arith(RuleName, Input, Expected, Class, Got) :-
    Goal =.. [RuleName, Input, Got],
    (   call_with_inference_limit(Goal, 10_000, Status)
    ->  (   Status == inference_limit_exceeded
        ->  Class = loop_detected
        ;   (Got =@= Expected -> Class = well_formed ; Class = wrong_answer)
        )
    ;   Class = undefined, Got = no_output
    ).

%! classify_entail(+Shape, +Target, +Claim, -Class) is det.
% Claim is `holds` or `fails`.
classify_entail(Shape, Target, holds, Class) :-
    ( entails_via_incompatibility(Shape, Target)
    -> Class = well_formed       % student correctly says it holds — dedup
    ;  Class = wrong_answer      % student says holds but axiom says it doesn't
    ).
classify_entail(Shape, Target, fails, Class) :-
    ( \+ entails_via_incompatibility(Shape, Target)
    -> Class = well_formed       % student correctly says it fails — dedup
    ;  Class = wrong_answer      % student says fails but axiom says it holds
    ).

%! run_all/0
run_all :-
    writeln('=== Misconception Test Harness ==='),
    run_arith_checks(ArithResults),
    run_entail_checks(EntailResults),
    append(ArithResults, EntailResults, All),
    length(All, Total),
    include([r(_,_,_,wrong_answer,_)]>>true, All, WA),
    include([r(_,_,_,loop_detected,_)]>>true, All, LD),
    include([r(_,_,_,undefined,_)]>>true, All, Undef),
    include([r(_,_,_,well_formed,_)]>>true, All, WF),
    length(WA, NWA), length(LD, NLD), length(Undef, NUndef), length(WF, NWF),
    format('~nResults: ~w total~n', [Total]),
    format('  wrong_answer:  ~w~n', [NWA]),
    format('  loop_detected: ~w~n', [NLD]),
    format('  undefined:     ~w~n', [NUndef]),
    format('  well_formed:   ~w  (check for dedup)~n', [NWF]),
    write_csv(All),
    writeln('Results written to misconceptions/results.csv').

run_arith_checks(Results) :-
    findall(
        r(Source, Domain, Desc, Class, Got),
        (   arith_misconception(Source, Domain, Desc, Rule, Input, Expected),
            classify_arith(Rule, Input, Expected, Class, Got)
        ),
        Results
    ).

run_entail_checks(Results) :-
    findall(
        r(Source, geometric, Desc, Class, entailment),
        (   entail_misconception(Source, Desc, Shape, Target, Claim),
            classify_entail(Shape, Target, Claim, Class)
        ),
        Results
    ).

write_csv(Rows) :-
    open('misconceptions/results.csv', write, S),
    format(S, 'source,domain,description,classification,got~n', []),
    forall(member(r(Src, Dom, Desc, Class, Got), Rows),
           format(S, '~w,~w,~w,~w,~w~n', [Src, Dom, Desc, Class, Got])),
    close(S).
```

% === G4Q1: Fraction ordering, butterfly variants ===
% Task: Order frac(2,3), frac(3,4), frac(3,8) smallest to largest.
% Correct: [frac(3,8), frac(2,3), frac(3,4)]

% --- Correct strategy (cross-multiply all pairs, assemble by wins) ---
g4q1_correct([frac(2,3), frac(3,4), frac(3,8)], [frac(3,8), frac(2,3), frac(3,4)]).

% --- Helper: cross_product/3 ---
% cross_product(frac(N1,D1), frac(N2,D2), first_greater|second_greater|equal)
cross_product(frac(N1,D1), frac(N2,D2), Result) :-
    C1 is N1 * D2,
    C2 is N2 * D1,
    % GROUNDED: multiply_grounded(RN1, RD2, Cross1), multiply_grounded(RN2, RD1, Cross2)
    % SCHEMA: Measuring Stick — numerators and denominators as lengths
    (C1 > C2 -> Result = first_greater
    ; C2 > C1 -> Result = second_greater
    ; Result = equal).

% Pairwise score: +1 if F beats Other, -1 if Other beats F
score(F, Pairs, Score) :-
    findall(D, pair_delta(F, Pairs, D), Ds),
    sumlist(Ds, Score).
pair_delta(F, Pairs, +1) :- member(pair(F,_,first_greater), Pairs).
pair_delta(F, Pairs, -1) :- member(pair(F,_,second_greater), Pairs).
pair_delta(F, Pairs, -1) :- member(pair(_,F,first_greater), Pairs).
pair_delta(F, Pairs, +1) :- member(pair(_,F,second_greater), Pairs).

% --- Variant 06-03: skips (2/3, 3/8) comparison, defaults first_greater ---
% CONNECTS TO: s(comp_nec(unlicensed(skip_pair(frac(2,3), frac(3,8)))))
g4q1_06_03([frac(2,3), frac(3,4), frac(3,8)], Order) :-
    cross_product(frac(2,3), frac(3,4), R1),
    cross_product(frac(3,4), frac(3,8), R2),
    % Skip: (2/3, 3/8) — default to first_greater without computing
    Pairs = [pair(frac(2,3), frac(3,4), R1),
             pair(frac(3,4), frac(3,8), R2),
             pair(frac(2,3), frac(3,8), first_greater)],
    Fracs = [frac(2,3), frac(3,4), frac(3,8)],
    maplist([F, S-F]>>(score(F, Pairs, S)), Fracs, Scored),
    keysort(Scored, Sorted),
    pairs_values(Sorted, Order).

% --- Variant 18-04: skips (2/3, 3/8), fills by magnitude transfer ---
% CONNECTS TO: s(comp_nec(unlicensed(magnitude_transfer(from_prior_pair,...))))
g4q1_18_04([frac(2,3), frac(3,4), frac(3,8)], Order) :-

    cross_product(frac(3,4), frac(3,8), R2),
    % Transfer: infer (2/3 > 3/8) from "2/3 was larger in the first pair"
    Pairs = [pair(frac(2,3), frac(3,4), R1),
             pair(frac(3,4), frac(3,8), R2),
             pair(frac(2,3), frac(3,8), second_greater)],  % wrong inference

% --- Variant 06-20: all pairs compared but (2/3, 3/4) uses mutated products ---
% CONNECTS TO: s(comp_nec(unlicensed(mutation(scale(8->24, 9->18)))))
g4q1_06_20([frac(2,3), frac(3,4), frac(3,8)], Order) :-

    % But student mutated: 8→24, 9→18, reversing the result.

## Task 5: DB Triage — Normalize and Export In-Scope Rows

**Files:**

- [ ] **Step 1: Write triage_export.py**

```python
#!/usr/bin/env python3
"""
Normalize topic casing and export in-scope rows to CSV files.

In-scope: fraction, decimal, whole_number, measurement, geometric domains
          AND (PST population OR K-5 grade band keywords)

Out-of-scope rows → misconceptions/out_of_frame.csv with reason tag.
In-scope rows     → misconceptions/corpus_inscope.csv (one per domain).

Run from repo root:
  python3 research_corpus/scripts/triage_export.py
"""

import sqlite3
import csv
import os

DB = "research_corpus/research.db"
OUT_INSCOPE = "misconceptions/corpus_inscope.csv"
OUT_OOF = "misconceptions/out_of_frame.csv"

IN_SCOPE_DOMAINS = {'fraction', 'decimal', 'whole_number', 'measurement', 'geometric'}

GRADE_KEYWORDS = [
    'elementary', 'primary', 'kindergarten', 'first', 'second', 'third', 'fourth', 'fifth',
    'grade 1', 'grade 2', 'grade 3', 'grade 4', 'grade 5', 'k-5', 'k-3', 'k-2',
    'grade k', 'grades 1', 'grades 2', 'grades 3', 'grades 4', 'grades 5',
]

def is_inscope_grade(grade_band, pop_type):
    if pop_type in ('student', 'PST'):
        return True
    if not grade_band:
        return False
    gb = grade_band.lower()
    return any(kw in gb for kw in GRADE_KEYWORDS)

def oof_reason(domain, grade_band, pop_type, error_desc):
    if domain not in IN_SCOPE_DOMAINS:
        return 'domain_out_of_scope'
    if not is_inscope_grade(grade_band, pop_type):
        return 'grade_out_of_scope'
    if not error_desc or len(error_desc.strip()) < 20:
        return 'too_vague'
    return None

def main():
    os.makedirs("misconceptions", exist_ok=True)
    conn = sqlite3.connect(DB)
    conn.row_factory = sqlite3.Row
    cur = conn.cursor()

    # Step 1: normalize topic casing
    cur.execute("UPDATE error_instances SET mathematical_topic = lower(mathematical_topic) "
                "WHERE mathematical_topic != lower(mathematical_topic)")
    conn.commit()
    print(f"Normalized {cur.rowcount} topic rows")

    # Step 2: fetch all rows with article metadata
    cur.execute("""
        SELECT e.id, e.error_description, e.example, e.mathematical_domain,
               e.mathematical_topic, e.mathematical_subtopic,
               e.orientation, e.salience, e.page_refs,
               a.population_type, a.grade_band_as_declared,
               a.bibtex_key, a.authors, a.year, a.journal,
               a.local_pdf_path
        FROM error_instances e
        JOIN articles a ON e.article_id = a.id
    """)
    rows = cur.fetchall()
    conn.close()

    inscope_fields = ['id', 'error_description', 'example', 'mathematical_domain',
                      'mathematical_topic', 'mathematical_subtopic',
                      'population_type', 'grade_band_as_declared',
                      'bibtex_key', 'authors', 'year', 'journal',
                      'orientation', 'salience', 'page_refs', 'local_pdf_path']
    oof_fields = inscope_fields + ['oof_reason']

    inscope_rows = []
    oof_rows = []

    for r in rows:
        reason = oof_reason(r['mathematical_domain'], r['grade_band_as_declared'],
                            r['population_type'], r['error_description'])
        if reason:
            row = dict(r) | {'oof_reason': reason}
            oof_rows.append(row)
        else:
            inscope_rows.append(dict(r))

    with open(OUT_INSCOPE, 'w', newline='') as f:
        w = csv.DictWriter(f, fieldnames=inscope_fields, extrasaction='ignore')
        w.writeheader()
        w.writerows(inscope_rows)

    with open(OUT_OOF, 'w', newline='') as f:
        w = csv.DictWriter(f, fieldnames=oof_fields, extrasaction='ignore')
        w.writeheader()
        w.writerows(oof_rows)

    # Summary by domain
    from collections import Counter
    domain_counts = Counter(r['mathematical_domain'] for r in inscope_rows)
    print(f"\nIn-scope: {len(inscope_rows)} rows → {OUT_INSCOPE}")
    for domain, n in sorted(domain_counts.items(), key=lambda x: -x[1]):
        print(f"  {domain}: {n}")
    print(f"\nOut-of-frame: {len(oof_rows)} rows → {OUT_OOF}")

if __name__ == '__main__':
    main()
```
