Architecture · v0.1

How Hermes thinks. Neuro plus symbolic, in a tight loop.

Hermes is two systems working together: an open-source language model that listens, and a deterministic Prolog core that reasons about strategies and standards. The loop between them — feed the symbolic skeleton a lesson plan and the LLM only has to ask "is this that?" — is what cuts inference cost enough to run on a teacher's laptop.

symbolic · the skeleton

Deterministic core · no slop, 2 always equals 2

Prolog for good-old-fashioned AI and pattern recognition. Each strategy is a horn-clause rule that produces a trace. Misconceptions are broken rules. The LLM's proposals only count once they unify with the symbolic side.

standardsCCSS & state frameworks, machine-readable graph
strategiesautomata for student-invented strategies (COBO · RMB · chunking · ...)
misconceptionsdocumented bugs from the math-ed literature, as Prolog clauses
incoherenceBrandom-style frames at scene-level (Carspecken, 1999)
prooftheorem proving — why two strategies are equivalent, or one contradicts

↻
The loop is the trick. Generic LLM transcript-coding is ruinously expensive — every word is a fresh hard problem. Hermes flips it: feed the symbolic core a lesson plan, and it already knows what to listen for. The LLM only has to ask "is this that?" against a small candidate set. Inference cost drops by orders of magnitude. That's what makes local-first deployment realistic.
deployment · privacy · cost

Where Hermes actually runs.

A live classroom is the highest-stakes data context in education. Hermes is designed so the most natural deployment is also the most private one: on the teacher's own machine. The neuro side is the only thing that has to be big, and we're working to shrink it.

now · prototype

REALLMs API · research platform

Indiana University's REALLMs platform hosts open-source models for researcher use. Audio & transcripts go through REALLMs; the symbolic core runs locally. IU IRB-friendly.

Gemma · on the laptop

A Gemma-class model — small enough for a teacher's laptop or a $400 mini-PC under the SmartBoard. Symbolic core stays local. Audio never leaves the room. The lesson-plan loop is what makes the small model enough.

School-server option · optional

For districts that want shared roster & standards data across rooms, a single school-level server runs the LLM, classrooms run the symbolic core. Still no audio leaves the building.
